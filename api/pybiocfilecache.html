pybiocfilecache.cache module¶

class pybiocfilecache.cache.BiocFileCache(cache_dir: str | Path | None = None, config: CacheConfig | None = None)[source]¶

Bases: object

Enhanced file caching module.

Features: - Resource validation and integrity checking - Cache size management - File compression - Resource tagging - Automatic cleanup of expired resources

add(rname: str, fpath: str | Path, rtype: Literal['local', 'web', 'relative'] = 'local', action: Literal['copy', 'move', 'asis'] = 'copy', tags: List[str] | None = None, expires: datetime | None = None, ext: bool = False) → Resource[source]¶

Add a resource to the cache.

Parameters:

  • rname – Name to identify the resource in cache.

  • fpath – Path to the source file.

  • rtype – Type of resource. One of local, web, or relative. Defaults to local.

  • action – How to handle the file (“copy”, “move”, or “asis”). Defaults to copy.

  • tags – Optional list of tags for categorization.

  • expires – Optional expiration datetime. If None, resource never expires.

  • ext – Whether to use filepath extension when storing in cache. Defaults to False.

Returns:

The Resource object added to the cache.

List resources in the cache with optional filtering.

Parameters:

  • rtype – Filter resources by type.

  • expired –

    Filter by expiration status

    Note: Resources with no expiration are always considered non-expired.
